# Vite

Aquest projecte correspon a l'activitat: <https://xtec.dev/python/vite/>

## Desenvolupament

Executa el servidor Python:

```sh
$ poetry shell
$ poetry install
$ fastapi dev server/main.py
```

Executa el client React:

```sh
$ bun install
$ bun run dev
```

Obre el navegador a <http://localhost:3000/>
